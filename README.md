# Secure Password Generator

A dynamic and interactive web application that allows users to generate secure, randomized passwords based on custom criteria.

## 🚀 Features
* **Custom Length:** Use the slider to set a password length between 1 and 20 characters.
* **Character Variety:** Options to include uppercase letters, lowercase letters, numbers, and special symbols.
* **Strength Indicator:** A visual color-coded indicator (Green, Yellow, Red) that evaluates the strength of the generated password based on its complexity.
* **Copy to Clipboard:** One-click functionality to copy the generated password with visual feedback.

## 🛠️ Tech Stack
* **HTML5:** Used for the structure of the generator interface.
* **CSS3:** Features a vibrant linear gradient background and interactive components like tooltips and indicators.
* **Vanilla JavaScript:** Powers the password logic, including the Fisher-Yates shuffle algorithm for maximum security.

## 📦 Installation
1. Clone the repository
2. Open index.html in your browser to start generating passwords!
